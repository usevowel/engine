# Speech-to-Text (STT)

## Supported Providers

| Provider | Features |
|----------|----------|
| `groq-whisper` | Groq Whisper Large V3, batch mode, high quality |

## Configuration

```bash
STT_PROVIDER=groq-whisper
```

## Provider Details

### Groq Whisper
- **Model:** Whisper Large V3
- **Mode:** Batch processing
- **Quality:** High
- **Latency:** Moderate
